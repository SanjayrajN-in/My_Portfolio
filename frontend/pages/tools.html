Image Scale Measurement

Measure objects in images using reference scale

Drop image here or click to upload

Instructions:

  1. Upload an image - Drag & drop or click to select
  2. Set Reference Scale - Click the button (or Ctrl+R) and draw a line on a known measurement
  3. Enter Reference Length - Input the actual length of your reference line
  4. Measure Objects - Click "Measure Object" (or Ctrl+M) and draw lines to measure
  5. View Results - Click on measurements in the list to highlight them
Keyboard Shortcuts:
• Ctrl+R: Set Reference Scale
• Ctrl+M: Measure Object
• Ctrl+C: Clear All
• Escape: Cancel current mode
